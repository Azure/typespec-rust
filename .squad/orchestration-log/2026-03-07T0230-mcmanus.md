```rust

mod tests {
    use super::*;

    const _: () = {
        fn _touch_internal_methods() {
            let _ = || {
                drop(internal_decorator_in_internal());
                drop(internal_method_in_operation());
                drop(internal_decorator_in_operation_async());

        }

```

This pattern makes all transitive types reachable without suppressing any warnings.

## Team Pattern Established

When a test crate has `pub(crate)` methods with no callers in test code:

1. **Do NOT suppress** with `#[allow(dead_code)]` at the emitter level
2. **Add exercising code** in the hand-written `lib.rs` using the `const _: () = { ... }` pattern
3. This validates that internal methods actually work without masking issues
